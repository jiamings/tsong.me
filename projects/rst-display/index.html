Rectilinear Steiner Tree

by Jiaming Song

Figure 1: RMST and LMST algorithms displayed with Qt.

Abstract

In this project, three methods (SMST, LMST, ZMST), which are proposed in this paper New Algorithms for the Rectilinear Steiner Tree Problem, are implemented. The implementation was selected for the 2012-2013 Summer Programming Course as a base algorithm for a Qt5 GUI project.
